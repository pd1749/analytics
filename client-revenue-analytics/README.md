# Client Revenue Analytics

Service performance and revenue tracking dashboard for business reporting.

## Summary

Built interactive Excel dashboard tracking $1.3M in client transactions across service categories and regions.

## Key Metrics

| KPI | Value |
|-----|-------|
| Total Revenue | $1,304,750 |
| Avg Order Value | $1,305 |
| Top Service | Mentoring (50% of revenue) |
| Cancellation Rate | 10.3% |

## Findings

- Mentoring program drives half of revenue despite quarter of orders
- NT region outperforms on revenue per client
- 10.3% cancellation rate = $129K recoverable
